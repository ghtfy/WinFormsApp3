## 📌 기능 구현

---

### 1️⃣ 버튼 클릭 이벤트 Random 클래스 활용

1.이벤트생성 ▶디자인창에서"배경색랜덤변경" 버튼을 더블클릭하여 코드 작성창으로 이동합니다.

2.로직작성 : 아래코드를 참조하여 버튼을 누를때마다 배경색이 무작위로 변하게 코딩합니다.
